# 04 — Tipos de Dados
**Objetivo.** Comparar tipagem em 3 linguagens.
**Sugestão de trio:** Python (dinâmica), JavaScript (dinâmica), C (estática).
**Exemplos mínimos.**
- **C** (estática): `int a = 10; a = "oi"; // erro`
- **Python** (dinâmica): `a = 10; a = "oi"  # ok`
- **JS** (dinâmica + coerção): `1 + "2" // "12"`
